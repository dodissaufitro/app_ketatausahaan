🆘 QUICK TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Problem: Connection refused
  → Cek X601_API_BASE_URL
  → Ping mesin X601: ping 192.168.1.100
  → Cek firewall & network

Problem: Employee not found
  → Ensure employee_id sama di sistem & X601
  → Check employee list di dashboard

Problem: Invalid API Key
  → Verify X601_API_KEY di .env
  → Request API key baru dari admin X601

Problem: Sync tidak ada data
  → Check logs: tail -f storage/logs/laravel.log
  → Verify X601 API format
  → Contact X601 support
